# Dataset

This folder contains the dataset used throughout the project.

The dataset provides movie metadata and descriptions that are used for data analysis, preprocessing, feature engineering, model training, and evaluation.

## Target Variable

- **Genre** – The movie genre to be predicted (e.g., Action, Comedy, Drama, Horror, Romance).

## Features

| Feature | Description |
|---------|-------------|
| Title | Movie title |
| Year | Release year |
| Director | Movie director |
| Duration | Runtime in minutes |
| Rating | Average movie rating |
| Votes | Number of user votes |
| Description | Movie synopsis used for text analysis |
| Language | Primary language |
| Country | Production country |
| Budget_USD | Production budget |
| BoxOffice_USD | Box office revenue |
| Production_Company | Production company |
| Content_Rating | Age rating |
| Lead_Actor | Main actor or actress |
| Num_Awards | Number of awards |
| Critic_Reviews | Number of critic reviews |

## Purpose

This dataset is used to:

- Perform Exploratory Data Analysis (EDA)
- Preprocess structured and text data
- Build a movie genre classification model
- Compare multiple machine learning algorithms
- Evaluate model performance
